# DVD-Sammlung Terminal App (MongoDB-Katalog)

Dieses Projekt ist eine terminalbasierte **DVD-Sammlungsanwendung**, die eine Verbindung zu einer **MongoDB-Datenbank** herstellt.

Die Anwendung wurde in **Python** entwickelt und nutzt **PyMongo** zur Kommunikation mit der Datenbank.

## Funktionen

- Verbindung zu einer MongoDB-Datenbank über PyMongo
- Abfragen von Datensätzen direkt im Terminal
- Hinzufügen und Bearbeiten von Daten in der Datenbank
- Verwaltung einer DVD-Sammlung über die Kommandozeile

## Technische Details

- Programmiersprache: Python
- Datenbank: MongoDB
- Bibliothek: PyMongo
